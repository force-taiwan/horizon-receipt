領據系統核心亮點功能

  • 多輪增量語意填表 (HITL)

    AI 小幫手採用記憶式增量合併演算法。您輸入或修改新欄位時，AI 只會針對新增的資料做回應與更新，而不會覆蓋掉您先前已經正確填寫完畢的欄位，實現自然的對話式填單體驗。

  • 手寫圖片多模態 OCR 解析

    支援直接拍照或上傳領據草稿、手寫便簽圖片。後端採用 Gemini 3.1 Flash-Lite 多模態視覺模型進行原生 OCR 識別，能精準提取手寫姓名、身分證字號、金額與門號等資訊。

  • 前端自動尺寸與品質壓縮

    在上傳手寫單據照片前，瀏覽器會自動在背景將大圖等比例收縮（長邊最大 1280px）並以 75% 品質轉換為 JPEG 格式。能有效節省 95% 以上的頻寬，降低辨識延遲並節省 API Token 折耗。

  • 智慧糾錯與預判斷機制

    自動預判手機 (09開頭) 與市話；並依據台灣地理資料自動糾正地名 typo 錯字（如永坑區糾正為永康區，苓牙區糾正為苓雅區），防範人為或語音輸入產生的低級錯誤。

  • 雙向稅率與扣繳額聯動計算

    輸入金額與身分資料後，系統能智慧判別所得類別（本國籍/外籍、執行業務/租金/薪資等），自動套用正確稅率並計算扣繳稅額與實領金額，提供人工填單覆檢雙向校對。
